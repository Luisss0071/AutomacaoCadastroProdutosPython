tipo, categoria, preco_unitario, custo e a obs de produtos
em uma base de dados, economizando horas de trabalho manual, 
através de um código em python, utilizando as seguintes bibliotecas:

- pandas - Ler a base de dados
- pyautogui - Automatizar o processo de cadastro
- time - Utilizado para pausar o sistema por x segundos
